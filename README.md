# README
## 設定方法
1. リポジトリ内のテキストファイルをダウンロード。
2. Google日本語入力とMozcの設定画面（プロファイル）を開き、「ローマ字テーブル」を開く。（編集）
3. 現在登録されているものを全て選択し、「編集」から、「選択されたエントリーを削除」。
4. 「編集」から「インポート」を押し、ダウンロードしたテキストファイルを選択。
5. 「適用」→「OK」

## 特徴
基本的な入力ルールは、エスペラントの発音規則に則ります。  
また、エスペラント特殊文字はすべてX方式に対応しています。
なお、実際の打鍵で違和感なく打てるように、いくつか独自の法則性があります。

### 独自の法則の例
- wa,wi,we,wo → わ、ゐ、ゑ、を
- wji,wje → ゐ、ゑ
- hu,fu → ふ、ふ
- h,hx → ハ行
- r,l → ラ行
- sx、sj → シャ行
- cx,cj→チャ行
- jxi,zxi,zji → じ
- cxi,cji → ち
- dzu,dxu → づ
- xa,xi,xu,xe,xo,xja,xji,xju,xje,xjo → ぁ、ぃ、ぅ、ぇ、ぉ、ゃ、ぃ、ゅ、ぇ、ょ
- uxa,uxi,uxu,uxe,uxo → うぁ、うぃ、うぅ、うぇ、うぉ
- xwa,xcu → ゎ、っ
